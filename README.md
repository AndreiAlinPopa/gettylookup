# GettyLookup
 Web scraper for Gettyimages. Takes a search term, and opens a desired amount of images in browser.
## How to setup
1. Firstly, make sure you have Python 3.7+ and pip installed.
2. Install the bs4 library. Go into terminal / cmd and input: 
```
pip install bs4
```
 3. Locate gettylookup.py. Run the .py file with Python and follow the instructions
## How to search
**Search:** Term to search on GettyImages

**Quantity:** Number of images to open in browser
